Instalar no iPhone

Esta versão funciona como aplicativo web instalável. Abra este link no Safari do iPhone e adicione o app à Tela de Início.

Abrir aplicativo Baixar pacote Versão Android

Passos no iPhone

Safari
  1. Abra esta página no Safari do iPhone.
  2. Toque no botão de compartilhar.
  3. Escolha “Adicionar à Tela de Início”.
  4. Confirme o nome “Time em Quadra”.

Observação

Para ter um link oficial da App Store, o app precisa ser convertido para projeto iOS, assinado com uma conta Apple Developer e enviado para revisão da Apple.
