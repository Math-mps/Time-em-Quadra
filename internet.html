Acessar aplicativo online

O aplicativo está pronto para ser publicado na internet. Depois da hospedagem, este botão pode apontar para o endereço público do time.

Abrir versão local Baixar site para publicar Link público pendente

Como deixar online

Publicação
  1. Enviar os arquivos da pasta do aplicativo para uma hospedagem.
  2. Publicar como site estático em um serviço como Netlify, Vercel, GitHub Pages ou servidor próprio.
  3. Copiar o link público gerado pela hospedagem.
  4. Substituir o botão “Link público pendente” pelo endereço final.

Arquivos necessários

Para colocar no ar, publique estes arquivos juntos: index.html, styles.css, app.js, manifest.webmanifest, ios.html, android.html e internet.html.

Link compartilhável

O link para terceiros será gerado pela plataforma de hospedagem depois que o pacote do site for publicado. O endereço local começando com file:/// funciona apenas neste computador.
