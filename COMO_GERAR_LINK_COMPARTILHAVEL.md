# Como gerar um link compartilhavel

O aplicativo esta pronto para publicacao como site estatico.

## Arquivo preparado

Use o pacote:

`time-em-quadra-site-publicavel.zip`

## Caminho rapido

1. Entre em uma plataforma de hospedagem de site estatico.
2. Envie o arquivo `time-em-quadra-site-publicavel.zip` ou os arquivos extraidos.
3. Aguarde a publicacao.
4. Copie o link publico gerado pela plataforma.
5. Compartilhe esse link com atletas, treinadores e terceiros.

## Observacao importante

Links que comecam com `file:///` funcionam apenas no computador onde os arquivos estao salvos. Para terceiros visualizarem, o aplicativo precisa estar publicado na internet.
